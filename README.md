# TankWar
## 坦克大战说明：
本坦克大战模仿马士兵尚学堂单机版坦克大战的思路，并在他版本的基础上进行改进，改进的点有：<br>
  >1.敌方坦克随机生成<br>
  >2.血块随机位置生成并且路径AI与敌方坦克路径AI相同<br>
  >3.血块虽大个数为3，由一个专门管理血块的线程进行生成新的血块<br>
## 操作方法：<br>
>1.上下左右键控制方向；<br>
>2.ctrl键进行单发炮弹的射出；<br>
  >3.A键可以发射超级炮弹，炮弹从八个方向射出；<br>
  >4.F12为复活键；<br>
  
  
  
  ps：玩的时候记得把输入法调成英文的。。
